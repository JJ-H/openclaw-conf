安装 Gitee Agent Skills

选择适合你的安装方式

推荐：让 AI Agent 自动安装

帮我安装 Gitee Agent Skills，
仓库地址是 https://gitee.com/oschina/gitee-agent-skills，找到 OpenClaw 相关安装指引并安装
# 1. 克隆 Gitee Agent Skills 仓库
git clone https://gitee.com/oschina/gitee-agent-skills
# 2. 创建 Skills 目录（如果不存在）
mkdir -p ~/.openclaw/skills
# 3. 复制到 OpenClaw Skills 目录
cp -r gitee-agent-skills/skills/* ~/.openclaw/skills/
# 4. 重启 OpenClaw 生效
openclaw gateway restart
