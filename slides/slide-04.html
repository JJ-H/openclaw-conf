什么是 Skills？

给 AI Agent 装上"手艺"，让它从"会聊天"进化到"会干活"

📋

Skill 是什么？

AI Agent 的能力扩展协议，定义了 Agent 如何理解和执行特定任务。

通过标准化的 Skill 定义，让 AI 能够：

  • 理解复杂任务意图
  • 编排多步骤工作流
  • 调用外部工具执行
  • 在多客户端复用
🎯

解决了什么问题？

💸 上下文膨胀

MCP 需注入整个 API 定义，Token 消耗巨大

🔄 数据搬运低效

模型被迫当"路由器"，中间结果重复耗 Token

🧩 工具流程割裂

MCP 只定义"手怎么动"，没定义"脑怎么想"

🔌

技术协议

Skills 采用开放协议，可在任何支持 Agent Skills 的应用中使用：

✓ OpenClaw / Claude Code
✓ Cursor / Gemini CLI
✓ OpenAI Codex / Goose
✓ 更多客户端持续接入...

核心价值： Skills 将工具调用、任务编排、领域知识封装为可复用的能力模块， 让开发者通过自然语言即可驱动 AI 完成复杂工作流。
